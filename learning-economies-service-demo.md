# learning economies demo

_this is highly incomplete, and needs to be expanded_

## overview

the aim is to make an online service that will coordinate agentic support to a teacher/educator, that is supporting their students.

## The students will need

- present learning direction / content
- tell the Learning Economies system information that determines which of the learning economies that are most aligned to them.

## the teacher will need

- tell the Learning Economies system about each/any of their students
- learn which of the learning economies the student is most aligned to, and which they think they want, and why
- tell the system enough information to align the student to one of the Testable Hypotheses in the Learning Economies platform
- by using a Learning Economies agent conversation, help the student develop a personalized education plan, that is aligned to (and supported by) on the Testable Hypotheses in the Learning Economies platform.
