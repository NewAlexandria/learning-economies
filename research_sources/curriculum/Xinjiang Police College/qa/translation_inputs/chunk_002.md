ZH: 5.3.4 投标人提供的货物，其货物的投标价即货到交货地点的交货价格，包括制造、组装该货物所使用的零部件及原材料已付的全部关税、销售税和其他税。

<!-- id: p000201 type: paragraph -->
ZH: 5.4  投标人应对投标货物提供完整的详细的书面说明。

<!-- id: p000202 type: paragraph -->
ZH: 5.5  报价超过最高限价的，该报价为无效报价，不进入商务评比。

<!-- id: p000203 type: paragraph -->
ZH: 5.6  评标委员会认为投标人的报价明显低于其他通过符合性审查投标人的报价，有可能影响产品质量或者不能诚信履约的，应当要求该投标人在评标系统合理的时间内提供成本构成说明，必要时提供相关证明材料；投标人不能证明其报价合理性的，评标委员会应当将其作为无效投标处理。

<!-- id: p000204 type: paragraph -->
ZH: 6、招标文件规定的技术文件

<!-- id: p000205 type: paragraph -->
ZH: 6.1  投标人须提交证明拟供货物和服务符合招标文件规定的文件资料，作为投标文件的一部分。

<!-- id: p000206 type: paragraph -->
ZH: 6.2  上述文件可以是文字资料、图纸和数据等，并提供规格、技术参数偏离表。

<!-- id: p000207 type: paragraph -->
ZH: 6.3  没有按要求提供资料或提供资料不完全的，将被视为对招标文件没有做出实质性响应，其风险由投标人自行承担。

<!-- id: p000208 type: paragraph -->
ZH: 6.4  投标人提供的设备必须是正规厂家生产的高质量产品，不能提供劣质三无产品（无厂址,无商标,无合格证）。

<!-- id: p000209 type: paragraph -->
ZH: 7、投标有效期

<!-- id: p000210 type: paragraph -->
ZH: 7.1  投标文件从投标之日起，投标有效期90天。

<!-- id: p000211 type: paragraph -->
ZH: 7.2  在特殊情况下，采购方可与投标人协商延长投标文件的有效期。

<!-- id: p000212 type: paragraph -->
ZH: 8、投标文件的签署规定

<!-- id: p000213 type: paragraph -->
ZH: 8.1  投标文件的页面必须用印刷体打印，提倡双面打印。

<!-- id: p000214 type: paragraph -->
ZH: 8.2  投标文件应清楚工整，一般不准修改。个别非实质性修改之处应由投标人的被授权人或法人代表签章。

<!-- id: p000215 type: paragraph -->
ZH: 8.3  投标文件应由法人代表或授权代表在规定的签章处逐一签署并加盖单位公章。所有投标人签字、法人代表签字、法人代表授权人签字和其它签字处必须加盖具有法律效力的投标人的印章后，投标文件方为有效。

<!-- id: p000216 type: paragraph -->
ZH: 8.4  电报、电话、传真、电子邮件等形式的投标概不接受。

<!-- id: p000217 type: paragraph -->
ZH: 9、投标文件的修改和撤销

<!-- id: p000218 type: paragraph -->
ZH: 9.1  投标人在递交投标文件后，可在规定的投标截止时间之前，对其投标文件以书面通知的形式进行修改或撤销。该通知须有投标代理人的签字，并得到采购方的确认。

<!-- id: p000219 type: paragraph -->
ZH: 9.2  投标人对投标文件修改的书面材料或撤消通知应按招标文件要求进行密封、标注和递交，并注明“修改投标文件”或“撤消投标”字样，修改或撤消的内容须按招标文件的要求签署、盖章，并作为投标文件和组成部分。

<!-- id: p000220 type: paragraph -->
ZH: 9.3  对投标文件修改的书面材料应于投标截止日前送达采购方，投标截止时间以后不得修改投标文件。

<!-- id: p000221 type: paragraph -->
ZH: 9.4  投标人不得在开标后至投标有效期满前撤销投标文件，否则采购方将没收其投标保证金。

ZH: 10、投标文件的密封和标记

<!-- id: p000224 type: paragraph -->
ZH: 10.1  投标人应在投标截止时间之前完成政采云系统中的标书上传步骤。

<!-- id: p000227 type: paragraph -->
ZH: （2） 与采购文件有重大偏离的投标文件；
