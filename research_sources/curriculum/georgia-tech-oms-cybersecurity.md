# Georgia Institute of Technology - Online Master of Science in Cybersecurity

## Program Overview
- **Institution**: Georgia Institute of Technology
- **Program**: Online Master of Science in Cybersecurity (OMSCS)
- **Format**: Fully online
- **Total Courses**: 10 courses
- **Duration**: 2-3 years
- **Tuition**: Under $12,000
- **URL**: https://info.pe.gatech.edu/oms-cybersecurity/

## Program Structure

The program is interdisciplinary and offers three specialized tracks, allowing students to tailor their studies to their career goals.

### Specialized Tracks

#### 1. Information Security Track
- **Focus**: Principles and practical techniques for securing computing systems and networks
- **Topics Include**:
  - Penetration testing
  - Network security
  - Cryptography
  - Advanced security analysis tools
  - Hands-on experience with security tools

#### 2. Cyber-Physical Systems Track
- **Focus**: Security of systems interfacing with the physical world
- **Topics Include**:
  - Critical infrastructure security
  - IoT device security
  - Industrial control systems (ICS) security
  - Security monitoring methodologies
  - Attack detection
  - Risk analysis
  - Intrusion response
  - Designing and maintaining resilient cyber-physical systems

#### 3. Policy Track
- **Focus**: Legal and regulatory aspects of cybersecurity
- **Topics Include**:
  - Privacy and compliance
  - Administrative tools for cybersecurity
  - Market-based security approaches
  - Public policy tools
  - Organizational, national, and international security policies
  - Ethical considerations

## Program Features
- Fully online format
- Designed for working professionals
- Interdisciplinary approach
- Deep understanding of policies, principles, and practical techniques for securing cyberspace

## Notes
- The Cyber-Physical Systems track is particularly relevant for OT security
- Program provides comprehensive coverage of both technical and policy aspects of cybersecurity
