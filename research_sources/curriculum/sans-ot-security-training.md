# SANS Institute - OT Security Training Programs

## Program Overview
- **Institution**: SANS Institute
- **Type**: Professional training and certification programs
- **Focus**: Industrial Control Systems (ICS) and Operational Technology (OT) Security
- **URL**: https://www.sans.org/industrial-control-systems/

## Key OT Security Courses

### Foundational Courses

#### ICS410: ICS/SCADA Security Essentials
- **Course Code**: ICS410
- **Focus**: Foundational knowledge of ICS/SCADA security
- **Target Audience**: Security professionals new to ICS/SCADA environments
- **Topics Covered**:
  - ICS/SCADA fundamentals
  - Security challenges in industrial environments
  - Risk assessment for ICS systems
  - Security architecture for industrial systems

#### ICS515: Active Defense and Incident Response
- **Course Code**: ICS515
- **Focus**: Incident response and active defense for ICS environments
- **Target Audience**: Security professionals responsible for ICS incident response
- **Topics Covered**:
  - Incident response procedures for ICS
  - Active defense strategies
  - Threat hunting in industrial environments
  - Recovery and business continuity

### Advanced Courses

#### FOR508: Advanced Incident Response, Threat Hunting, and Digital Forensics
- **Course Code**: FOR508
- **Focus**: Advanced digital forensics and incident response
- **Relevance**: Applicable to OT environments for forensic analysis

#### FOR572: Advanced Network Forensics
- **Course Code**: FOR572
- **Focus**: Network forensics and analysis
- **Relevance**: Network security analysis for OT environments

### Certifications

#### GICSP: Global Industrial Cyber Security Professional
- **Certification**: GICSP (Global Industrial Cyber Security Professional)
- **Focus**: Comprehensive ICS/SCADA security certification
- **Requirements**: Typically requires completion of ICS410 and additional training
- **Coverage**:
  - ICS security fundamentals
  - Risk management
  - Security architecture
  - Incident response
