Each question includes:
- `question_text` - The question
- `correct_answer` - The right answer
- `distractors` - Wrong answers with misconception explanations
- `detailed_explanation` - Core logic, distractor analysis, analogy
- `difficulty` - foundational/intermediate/advanced
- `question_type` - definition/attack/defense/scenario/etc.
- `prerequisites` - Required knowledge
- `code_snippets` - Code examples (where applicable)
